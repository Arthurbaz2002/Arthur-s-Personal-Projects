# Arthur's Personal Projects

A collection of data science and machine learning projects, built to learn by doing and to apply analytics and AI tools to things I'm genuinely curious about, from aviation to sports to business analytics.

Each project lives in its own folder with a full writeup: what I built, what I tried, what worked, and what didn't. I use Claude (Anthropic) as an AI assistant throughout these projects, for debugging, explaining concepts, and reasoning through modeling decisions, alongside hands on coding and experimentation.

## Projects

**[Aircraft Manufacturer Classifier](./aircraft-classifier)**
A computer vision model that identifies an aircraft's manufacturer from a photo, using transfer learning with PyTorch. Final result: 89.5% test accuracy across 30 manufacturer classes, reached through several rounds of real experimentation with model architecture and training strategy.

More projects coming soon.
